**Release date:** 2026-07-15

## Summary

This release establishes the first public baseline of the Governance, Authority and Assurance Metamodel. It provides a complete architecture and initial normative requirements for authority, delegation, evidence, assurance, trust decisions, runtime governance, agentic systems, trust graphs, registries, accountability and redress.

## Included

- substantive normative specification;
- seven composable conformance profiles;
- requirements, relationship and conformance matrices;
- architecture, implementation and conformance guidance;
- initial examples and diagrams;
- TSMM v0.22.0 adoption crosswalk;
- open questions and milestone roadmap;
- repository governance, contribution and security documentation.

## Maturity

This is an Initial Public Draft. Normative concepts and identifiers may change before v1.0.0. Experimental implementation and detailed review are encouraged. Production claims should clearly identify the draft status and implementation-specific risk controls.

## Repository description

A vendor-neutral, protocol-independent metamodel for governing authority, delegation, accountability, evidence and trust decisions across distributed digital systems. Defines concepts, relationships, conformance profiles and governance semantics for interoperable trust architectures, agentic systems and decentralized trust graphs.
